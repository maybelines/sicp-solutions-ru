### Упражнение 1.10.
Рассмотрим процедуру, реализующую функцию Аккермана:
```scheme
(define (A x y)
    (cond ((= y 0) 0)
          ((= x 0) (* 2 y))
          ((= y 1) 2)
          (else (A (- x 1) 
                (A x (- y 1))))))
```
```
(A 1 10)
(A 0 (A 1 9))
(A 0 (A 0 (A 1 8)))
(A 0 (A 0 (A 0 (A 1 7))))
...
(A 0 (A 0 (... (A 1 1))...)
(A 0 (A 0 (... 2)...)
(A 0 2^9)
2^10
```
то есть $(A \; 1 \; x) = 2^x$
```
(A 2 4)
(A 1 (A 2 3))
(A 1 (A 1 (A 2 2)))
(A 1 (A 1 (A 1 (A 2 1))))
(A 1 (A 1 (A 1 2)))
(A 1 (A 1 2^2))
(A 1 2^2^2)
2^2^2^2
```
то есть $\def\rddots#1{\cdot^{\cdot^{\cdot^{#1}}}}
$ 2^{2^{\rddots2}} (A \; 2 \; x) = 2^{2^{\ldots^2}}$ &mdash; операция возведения в степень.
